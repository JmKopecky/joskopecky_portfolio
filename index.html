MathMinds Test Generation Software

Java desktop program that utilizes a JSON file of template questions to generate an output test pdf. Fields and selected questions are randomized to ensure a unique test every time. Additionally, the output test can include automatically generated multiple choice problems using a class of user-defined solver methods that take in the fields of a question and return an output.

Palaestra Tournament Software

Java web server that makes the job of conducting a tournament far easier by using a web interface over LAN. Compatible with a variety of bracket types, and allows substantial customization of scoring and conducting matches. Tournament managers can upload tests for each match, and each match is again conducted over LAN though a simple web interface.
